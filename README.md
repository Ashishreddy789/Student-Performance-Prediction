# Student Performance Prediction System

## Objective
To predict student academic performance (Pass/Fail) based on factors like attendance, study hours, and internal marks to identify students at risk of failing early.

## Tech Stack
* Language: Python
* Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
* Environment: VS Code / Jupyter Notebook

## Methodology
1. Data Collection from CSV records.
2. Data Preprocessing (handling missing values and encoding).
3. Exploratory Data Analysis (EDA) through visualization.
4. Training a Random Forest Classifier.
5. Evaluation using Accuracy and Confusion Matrix.

## Expected Outcome
Helps institutions take preventive actions by identifying weak students early through data-driven insights[cite: 1].
